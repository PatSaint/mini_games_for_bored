# Mini Games for Bored

Static neon arcade hub by **PatSaint**.

Open `index.html` to play locally, or publish the repository with GitHub Pages.

## Games

- Neon Snake
- Neon Tetris
- Neon Pong
- Space Invaders
- Buscaminas
- Flappy Neon
- Neon Breakout
- Neon 2048
- Neon Asteroids
- Neon Simon

## Current storage behavior

Each game stores its Top 10 locally using the browser's `localStorage`.

That means scores are saved only on the current browser/device. If the user clears site data, the local rankings can be lost.

## Publish with GitHub Pages

This project is ready for GitHub Pages because it has an `index.html` at the repository root and does not require a build step.

Recommended Pages source:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/root`
