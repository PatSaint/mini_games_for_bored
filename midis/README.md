# MIDI Sources

Source used for the downloadable MIDIs in this project:

- **CC0-midis** — https://github.com/m-malandro/CC0-midis
- License: **CC0 1.0 Universal**

Download script:

- `download-midis.ps1`

Suggested mapping:

- Snake → `midis/snake/maybe.mid`
- Tetris → `midis/tetris/gears.mid`
- Pong → `midis/pong/movin-on.mid`
- Space Invaders → `midis/invaders/frantic-boss-battle.mid`
- Breakout → `midis/breakout/lighthearted-battle-theme.mid`
- Flappy → `midis/flappy/do-you-remember.mid`
- Asteroids → `midis/asteroids/android-observation-room.mid`
- Simon → `midis/simon/gather-your-party.mid`
- Game of Life → `midis/life/sitar-jam.mid`
- Minesweeper → `midis/minesweeper/mine-all-mine.mid`
