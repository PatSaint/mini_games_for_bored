# Generated MIDI Pack

These MIDIs were generated locally to cover per-action cues for the arcade games.

Use `generate-midi-pack.ps1` to rebuild them at any time.

Suggested usage:
- `snake/start.mid`, `snake/eat.mid`, `snake/death.mid`
- `tetris/*`
- `pong/*`
- `invaders/*`
- `minesweeper/*`
- `flappy/*`
- `breakout/*`
- `life/*`
- `asteroids/*`
- `simon/*`
